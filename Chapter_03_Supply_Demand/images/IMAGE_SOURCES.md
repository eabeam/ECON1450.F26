1. **Download from Unsplash or Pexels** (both offer free, commercial-use images):
   - Go to unsplash.com or pexels.com
   - Search using suggested terms above
   - Download high-resolution versions
   - Rename to match filenames above

2. **Place images in this directory:**

3. **Attribution** (if required by license):
   - Unsplash: No attribution required but appreciated
   - Pexels: No attribution required
   - If using other sources, add attribution to presentation notes
